На сервере храниться некоторый справочник объектов в виде таблицы формата CSV. В таблице - основные атрибуты объектов. Одно из свойств - в отдельных файлах. Например:
автор; название книги; жанр; имя файла с текстом;
преподаватель; должность; дисциплины; группы; файл с расписанием;
Создать HTML-страницу для просмотра справочника. В основном поле при просмотре справочника выдается краткая информация (автор, название). В дополнительном - информация отображается по требованию. Доступ к дополнительной информации из файлов осуществляется с помощью функций Ajax.
Часть 1
С помощью библиотеки DataTables построить отображение справочника на основе примеров из https://datatables.net/examples/index:
1) базовые возможности ( Basic initialisation ) (17 пунктов);
2) дополнительные возможности ( Advanced initialisation ) (16 пунктов);
3) использовать различные источники данных ( Data sources ) (4 пункта);
4) действия с таблицей ( API ) (13 пунктов);
5) использовать возможности ( Ajax ) (9 пунктов).
