Номер Функция jQuery Описание Пример
1 addClass() Добавляет класс к указанным объектам. Старые классы не уничтожаются.
2 offsetParent() Возвращает ближайшего родителя.
3 :image Возвращает элемент input с типом image.
4 outerWidth() Получает текущий outerWidth.
5 html() Возвращает содержание HTML.

6 children() Возвращает список детей объекта.
7 one() Добавляет обработчик события который будет вызван единожды.
8 mousedown() Устанавливет обработчик события на непосредственное нажатие кнопки мыши.
9 change() Устанавливает обработчик на события change. Применимо только к типу input. Является сокращением для метода on("change", fun).
10 animate() Позволяет запустить анимацию свойств CSS.
